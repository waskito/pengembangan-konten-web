Mengenai Tugas 1

Tugas pertama matakuliah Pengembangan Konten Web adalah membaca buku, namun ini adalah tugas praktik yang pertama. Tujuan dari tugas ini adalah mahasiswa dapat menuliskan tag - tag yang ada di dalam standar HTML. Di dalam halaman ini saya menambahkan tag - tag terbaru versi HTML5

Tag di dalam <head></head>

Tag <head></head> adalah kepala dalam dokumen html, tag ini tidak ditampilkan di browser, namun menampung informasi yang dikenali browser, berikut tag - tag di dalam <head></head> berikut penjelasannya:

  • <title>Contoh Judul Halaman</title> : Untuk memberi judul yang akan tampil di tab Browser. Spec untuk Tag Title
  • <meta> : Berisi informasi yang beragam, seperti character set, title, keywords, author, viewport, description, open graph . Spec untuk Tag Meta
  • <link href="/css/tugas1.css" rel="stylesheet"> : Digunakan untuk import css/stylesheet eksternal, bisa juga untuk import favicon. Spec untuk Tag Link
  • <base href="/"> : Untuk menspesifikasikan base URL / target di dalam dokumen Spec untuk Tag Base
  • <script></script> : Tag untuk menuliskan kode javascript ke dalam dokumen. Untuk pendekatan modern, menaruh script di dalam tag <head></head> tidak direkomendasikan, karena memperlambat initial loading halaman. Spec untuk Tag Script

Tag di dalam <body></body>

Tag <body></body> berisi konten yang akan ditampilkan ke user. Dan ada banyak sekali tag - tag yang dapat dipakai di dalam tag <body></body> ini, apalagi versi HTML5. Berikut beberapa yang sering dipakai di dalam tag <body></body>:

Heading 1 <h1></h1> Heading 1 sebagai judul teks yang terbesar

Heading 2 <h2></h2> Heading 2 sebagai judul teks kedua terbesar

Heading 3 <h3></h3> Heading 3 sebagai judul teks ketiga terbesar

Heading 4 <h4></h4> Heading 4 sebagai judul teks keempat terbesar

Heading 5 <h5></h5> Heading 5 sebagai judul teks kelima terbesar
Heading 6 <h6></h6> Heading 6 sebagai judul teks keenam terbesar

Tag - tag Paragraf

<p></p> Paragraf merupakan tag yang mengindikasikan teks tersebut merupakan satu paragraf, secara default akan memiliki style display:block di tiap blok kode.

Jika ingin ganti baris di dalam paragraf bisa menggunakan <br> (Break). Untuk menampilkan link, bisa menggunakan <a href=""></a>.
